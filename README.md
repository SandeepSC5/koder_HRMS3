# HRMS Dashboard — Timesheet & Attendance Tracker

This project is part of a modular **Human Resource Management System (HRMS)** designed to help organizations manage employee resources efficiently. It includes core features like **Timesheet submission**, **Leave application** and **Payrol Management**.

##  Modules Implemented

 **Module 1: Timesheet Submission**  
 **Module 2: Payroll Management**  
 Modules 3–5 are in progress and will be added in future commits.

## Tech Stack

- **Frontend**: HTML, CSS (Limelight font, responsive design)
- **Backend**: Java Spring Boot (REST APIs)
- **Database**: MySQL
- **Security**: Spring Security (planned for login/registration)
- **Build Tool**: Maven
